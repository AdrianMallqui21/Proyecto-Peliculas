# Proyecto-Peliculas 
---
Aplicación web para la gestión y registro de películas, desarrollada íntegramente en Python utilizando Streamlit. Este proyecto unifica la lógica de backend y la interfaz de usuario en un solo flujo de trabajo, permitiendo un despliegue rápido y funcional en entorno local.

## Características Técnicas
Arquitectura: Implementación de Programación Orientada a Objetos (POO) mediante el uso de clases, métodos y funciones modulares.

Base de Datos: Gestión de persistencia de datos utilizando SQL.

Framework: Streamlit para una interfaz interactiva y dinámica.

Desarrollo: Proceso de depuración y optimización de errores críticos asistido por modelos de lenguaje (Claude AI).

## Objetivo
El objetivo principal fue explorar el ecosistema de frameworks de Python, logrando crear un sitio web funcional que procesa lógica compleja sin necesidad de lenguajes de frontend adicionales.
